# Scanner


# Installation with Git Clone:

1. Git Clone :
   <pre><code>git clone https://github.com/et1909/Scanner</code></pre>

2. Head to the Scanner Directory:
   <pre><code>cd Scanner</code></pre>

3. Run main.go:
   <pre><code>go run main.go</code></pre>
